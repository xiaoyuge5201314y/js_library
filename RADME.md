# 本地开发
```
git clone至本地
npm install
src下新建文件编写
```
# 打包编译
npm run build
# npm 发布
```
1.更新package.json的version
2.copy根目录下的package.json和README.md至dist目录下
3.cd dist
4.npm login
5.npm publish --access=public
```
